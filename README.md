# Vehicle Tracking Challenge

Technical challenge developed for a Full Stack Developer position.

The project simulates a vehicle tracking platform with:

- Vehicle management API
- Realtime telemetry simulation
- Interactive map visualization
- Frontend and backend separation
- Dockerized environment

## Projects

### Backend API
- FastAPI
- PostgreSQL
- SQLAlchemy
- Celery
- Docker

Repository:
https://github.com/vehicle-tracking-challenge/vehicle-tracking-api

---

### Frontend Web
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Leaflet

Repository:
https://github.com/vehicle-tracking-challenge/vehicle-tracking-web

## Features

- Vehicle CRUD
- Interactive map visualization
- Simulated realtime updates
- Responsive interface
- API documentation with Swagger
